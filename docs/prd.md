# PRD - Marketing AI Report MVP

## Objetivo
Receber planilhas (CSV/XLSX), dados exportados de redes sociais e atividades do Trello para gerar relatório executivo automático com insights e recomendações.

## Escopo MVP
- Upload de arquivos
- Normalização para schema canônico
- Cálculo de KPIs
- Relatório em HTML/PDF
- Dashboard básico com filtros

## KPIs iniciais
CTR, CPC, CPL, CPA, CVR, ROAS

## Critérios de aceite
- Processar arquivos heterogêneos
- Exibir erros acionáveis por coluna/linha
- Gerar relatório em PT-BR
- Sem travamento com volume alto (virtualização/paginação)
