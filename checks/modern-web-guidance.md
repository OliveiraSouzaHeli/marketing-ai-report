# Checklist Modern Web Guidance

## Compatibilidade
- [ ] Chrome/Edge/Firefox/Safari (estáveis recentes)
- [ ] Fallback para APIs com suporte parcial

## Acessibilidade (WCAG AA)
- [ ] Navegação por teclado
- [ ] Labels/ARIA corretos
- [ ] Foco visível/contraste adequado

## Performance
- [ ] Carregamento incremental
- [ ] Sem bloqueio longo da main thread
- [ ] Virtualização para tabelas grandes

## Segurança
- [ ] Validação/sanitização de entrada
- [ ] Limite de upload por tipo/tamanho
- [ ] Sem segredos no client
